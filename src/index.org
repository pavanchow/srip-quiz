* Instructions to run the quiz
  *STEP 1 :* Clone the repository
  #+BEGIN_EXAMPLE
  git clone https://github.com/virtual-labs/srip-quiz.git
  #+END_EXAMPLE

  *STEP 2 :* Change directory to quiz repository
  #+BEGIN_SRC 
  cd srip-quiz
  #+END_SRC
  
  *STEP 3 :* Cleans the build if exists and then builds the
   quiz sources
  #+BEGIN_EXAMPLE
  make -k all
  #+END_EXAMPLE

  *STEP 4 :* Run the quiz server
  #+BEGIN_EXAMPLE
  make -k run
  #+END_EXAMPLE

  *STEP 5 :* To view the quiz in browser
  #+BEGIN_EXAMPLE
  http://localhost:8000/quiz.html
  #+END_EXAMPLE

  To remove the build 
  #+BEGIN_EXAMPLE
  make -k clean
  #+END_EXAMPLE
